pwd: Print Working Directory. Use this to see exactly where you are in your computer's folders.

ls: List. Shows all files and folders in your current location.

cd <folder>: Change Directory. Navigate into a specific folder.

cd .. moves you up one level.

mkdir <name>: Make Directory. Creates a new folder (e.g., mkdir shiplio-v2).

touch <file>: Creates a new, empty file (e.g., touch index.js).

rm <file>: Removes a file. Warning: This is permanent and does not go to the trash bin.

rm -rf <folder>: Forcefully removes a folder and everything inside it.mv <source> <destination>: Move or rename a file or folder.

cp <source> <destination>: Copy a file. Use `cp -r` for folders.

cat <file>: Concatenate and display the content of a file in the terminal.

clear: Clears the terminal screen for a fresh start.

code .: Opens the current folder in Visual Studio Code.

grep "<search_term>" <file>: Searches for a specific string within a file.

history: Shows a list of previously executed commands.

# Git Basics

git init: Initializes a new Git repository in the current folder.

git status: Shows the state of the working directory and the staging area.

git add .: Adds all changes in the current directory to the staging area.

git commit -m "<message>": Records the changes in the repository with a descriptive message.

git push: Sends local commits to a remote repository.
